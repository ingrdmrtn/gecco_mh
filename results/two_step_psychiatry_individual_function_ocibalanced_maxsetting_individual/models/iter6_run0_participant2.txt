Here are three new cognitive models for the two-step decision task.

### Model 1: Asymmetric Q-Learning with Decay and Stickiness
This model is a "pure" Model-Free learner that updates values based on prediction errors. It extends the previous best model by introducing **asymmetric learning rates** for positive and negative prediction errors (e.g., learning more from wins than losses, or vice versa). It retains the **decay** mechanism (forgetting unchosen options) and **stickiness** (perseveration), which were effective in previous iterations.

```python
def cognitive_model1(action_1, state, action_2, reward, model_parameters):
    """
    Asymmetric Q-Learning with Decay and Stickiness.
    
    Model-Free agent that learns from positive and negative prediction errors 
    at different rates. Includes decay for unchosen options and choice stickiness.
    
    Parameters:
    alpha_pos: [0, 1] Learning rate for positive prediction errors.
    alpha_neg: [0, 1] Learning rate for negative prediction errors.
    decay_rate: [0, 1] Decay rate for unchosen options.
    beta: [0, 10] Inverse temperature.
    stickiness: [0, 5] Choice perseverance bonus.
    """
    alpha_pos, alpha_neg, decay_rate, beta, stickiness = model_parameters
    n_trials = len(action_1)
    
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    log_loss = 0.0
    eps = 1e-10
    last_action_1 = -1
    
    for trial in range(n_trials):
        # Stage 1 Choice
        q_net_1 = q_stage1_mf.copy()
        if last_action_1 != -1:
            q_net_1[last_action_1] += stickiness
            
        exp_q1 = np.exp(beta * q_net_1)
        probs_1 = exp_q1 / np.sum(exp_q1)
        log_loss -= np.log(probs_1[action_1[trial]] + eps)
        
        last_action_1 = action_1[trial]
        state_idx = state[trial]
        
        if action_2[trial] != -1:
            # Stage 2 Choice
            exp_q2 = np.exp(beta * q_stage2_mf[state_idx])
            probs_2 = exp_q2 / np.sum(exp_q2)
            log_loss -= np.log(probs_2[action_2[trial]] + eps)
            
            # Stage 2 Update (Reward Prediction Error)
            pe_2 = reward[trial] - q_stage2_mf[state_idx, action_2[trial]]
            lr_2 = alpha_pos if pe_2 >= 0 else alpha_neg
            q_stage2_mf[state_idx, action_2[trial]] += lr_2 * pe_2
            
            # Stage 1 Update (TD Prediction Error)
            target_1 = q_stage2_mf[state_idx, action_2[trial]]
            pe_1 = target_1 - q_stage1_mf[action_1[trial]]
            lr_1 = alpha_pos if pe_1 >= 0 else alpha_neg
            q_stage1_mf[action_1[trial]] += lr_1 * pe_1
            
            # Decay unchosen options
            q_stage2_mf[state_idx, 1-action_2[trial]] *= (1 - decay_rate)
            q_stage2_mf[1-state_idx, :] *= (1 - decay_rate)
            q_stage1_mf[1-action_1[trial]] *= (1 - decay_rate)
            
    return log_loss
```

### Model 2: Hybrid Model with Separate Learning Rates and Stickiness
This model posits that the participant uses both Model-Based (planning) and Model-Free learning, but learns at different speeds for the two stages. It uses **separate learning rates** for the Spaceship choice (Stage 1) and the Alien choice (Stage 2). This accounts for the possibility that the participant tracks the slowly changing alien values (`lr_s2`) differently from how they update the value of the spaceships (`lr_s1`).

```python
def cognitive_model2(action_1, state, action_2, reward, model_parameters):
    """
    Hybrid Model with Separate Learning Rates and Stickiness.
    
    Combines Model-Based and Model-Free learning with distinct learning rates
    for the first (spaceship) and second (alien) stages.
    
    Parameters:
    lr_s1: [0, 1] Learning rate for Stage 1 (Spaceships).
    lr_s2: [0, 1] Learning rate for Stage 2 (Aliens).
    beta: [0, 10] Inverse temperature.
    w: [0, 1] Weight for Model-Based control (0=MF, 1=MB).
    stickiness: [0, 5] Choice perseverance bonus for Stage 1.
    """
    lr_s1, lr_s2, beta, w, stickiness = model_parameters
    n_trials = len(action_1)
    
    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    log_loss = 0.0
    eps = 1e-10
    last_action_1 = -1
    
    for trial in range(n_trials):
        # MB Calculation
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        # Net Q-values
        q_net_1 = w * q_stage1_mb + (1 - w) * q_stage1_mf
        if last_action_1 != -1:
            q_net_1[last_action_1] += stickiness
            
        exp_q1 = np.exp(beta * q_net_1)
        probs_1 = exp_q1 / np.sum(exp_q1)
        log_loss -= np.log(probs_1[action_1[trial]] + eps)
        
        last_action_1 = action_1[trial]
        state_idx = state[trial]
        
        if action_2[trial] != -1:
            # Stage 2 Choice
            exp_q2 = np.exp(beta * q_stage2_mf[state_idx])
            probs_2 = exp_q2 / np.sum(exp_q2)
            log_loss -= np.log(probs_2[action_2[trial]] + eps)
            
            # Updates
            pe_2 = reward[trial] - q_stage2_mf[state_idx, action_2[trial]]
            q_stage2_mf[state_idx, action_2[trial]] += lr_s2 * pe_2
            
            target_1 = q_stage2_mf[state_idx, action_2[trial]]
            pe_1 = target_1 - q_stage1_mf[action_1[trial]]
            q_stage1_mf[action_1[trial]] += lr_s1 * pe_1
            
    return log_loss
```

### Model 3: Hybrid Model with Decay and Stickiness
This model combines the full **Hybrid (MB/MF)** architecture with the **Decay** mechanism found to be effective in the pure MF model. It tests the hypothesis that the participant plans using the transition matrix (`w`) but still suffers from forgetting unchosen options (`decay_rate`). This combination of MB planning, MF learning, decay, and stickiness represents a comprehensive "supermodel" of the observed mechanisms.

```python
def cognitive_model3(action_1, state, action_2, reward, model_parameters):
    """
    Hybrid Model with Decay and Stickiness.
    
    Integrates Model-Based planning with a Model-Free system that exhibits 
    memory decay for unchosen options, plus choice stickiness.
    
    Parameters:
    learning_rate: [0, 1] Update rate for Q-values.
    decay_rate: [0, 1] Decay rate for unchosen options.
    beta: [0, 10] Inverse temperature.
    w: [0, 1] Weight for Model-Based control.
    stickiness: [0, 5] Choice perseverance bonus.
    """
    learning_rate, decay_rate, beta, w, stickiness = model_parameters
    n_trials = len(action_1)
    
    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    log_loss = 0.0
    eps = 1e-10
    last_action_1 = -1
    
    for trial in range(n_trials):
        # MB Calculation
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        # Net Q-values
        q_net_1 = w * q_stage1_mb + (1 - w) * q_stage1_mf
        if last_action_1 != -1:
            q_net_1[last_action_1] += stickiness
            
        exp_q1 = np.exp(beta * q_net_1)
        probs_1 = exp_q1 / np.sum(exp_q1)
        log_loss -= np.log(probs_1[action_1[trial]] + eps)
        
        last_action_1 = action_1[trial]
        state_idx = state[trial]
        
        if action_2[trial] != -1:
            # Stage 2 Choice
            exp_q2 = np.exp(beta * q_stage2_mf[state_idx])
            probs_2 = exp_q2 / np.sum(exp_q2)
            log_loss -= np.log(probs_2[action_2[trial]] + eps)
            
            # Updates
            pe_2 = reward[trial] - q_stage2_mf[state_idx, action_2[trial]]
            q_stage2_mf[state_idx, action_2[trial]] += learning_rate * pe_2
            
            pe_1 = q_stage2_mf[state_idx, action_2[trial]] - q_stage1_mf[action_1[trial]]
            q_stage1_mf[action_1[trial]] += learning_rate * pe_1
            
            # Decay unchosen
            q_stage2_mf[state_idx, 1-action_2[trial]] *= (1 - decay_rate)
            q_stage2_mf[1-state_idx, :] *= (1 - decay_rate)
            q_stage1_mf[1-action_1[trial]] *= (1 - decay_rate)
            
    return log_loss
```
