Here are three cognitive models that incorporate the participant's high OCI (Obsessive-Compulsive Inventory) score into the decision-making process.

### Model 1: OCI-Modulated Habit Formation (Model-Free Bias)
This model posits that individuals with high OCI scores rely more heavily on habit-based (model-free) learning rather than goal-directed (model-based) planning. The `w` parameter, which usually governs the balance between Model-Based (MB) and Model-Free (MF) control, is modulated by the OCI score. A higher OCI score pushes the weight `w` towards 0 (pure model-free), reflecting a tendency towards repetitive, habitual behavior often seen in compulsion.

```python
def cognitive_model1(action_1, state, action_2, reward, oci, model_parameters):
    """
    Model 1: OCI-Modulated Habit Formation.
    
    This model assumes that higher OCI scores lead to a stronger reliance on Model-Free (habitual)
    learning over Model-Based (goal-directed) planning. The mixing weight 'w' is dynamically 
    adjusted by the OCI score.
    
    Parameters:
    learning_rate: [0, 1] - Rate at which Q-values are updated.
    beta: [0, 10] - Inverse temperature for softmax choice rule.
    w_base: [0, 1] - Baseline weighting for Model-Based control (1=Pure MB, 0=Pure MF).
    
    The effective 'w' is calculated as: w_base * (1 - oci_score).
    High OCI reduces the effective w, increasing model-free influence.
    """
    learning_rate, beta, w_base = model_parameters
    n_trials = len(action_1)
    oci_score = oci[0] # Scalar value 0.733...
    
    # Calculate effective mixing weight based on OCI
    # High OCI reduces w_effective, biasing towards Model-Free (0)
    w_effective = w_base * (1.0 - oci_score)
    
    # Transition matrix (fixed for this task structure)
    # Row 0: Spaceship A -> [70% Planet X, 30% Planet Y]
    # Row 1: Spaceship U -> [30% Planet X, 70% Planet Y]
    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    
    # Q-values
    q_stage1_mf = np.zeros(2)       # Model-free values for stage 1 (Spaceships)
    q_stage2_mf = np.zeros((2, 2))  # Model-free values for stage 2 (Aliens per Planet)
    
    for trial in range(n_trials):
        # --- Stage 1 Choice ---
        
        # Model-Based Value Calculation:
        # V(state) = max(Q_stage2(state, action))
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        # Q_MB = Transition_Matrix * V(next_states)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        # Hybrid Value: Q_net = w * Q_MB + (1-w) * Q_MF
        q_net = w_effective * q_stage1_mb + (1 - w_effective) * q_stage1_mf
        
        # Softmax Policy
        exp_q1 = np.exp(beta * q_net)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        # Store probability of the chosen action
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        
        # --- Stage 2 Choice ---
        
        s_idx = int(state[trial]) # The planet arrived at
        
        # Standard Q-learning at second stage
        exp_q2 = np.exp(beta * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Learning / Updating ---
        
        r = reward[trial]
        
        # Prediction errors
        # Stage 2 PE: Reward - Expectation
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        
        # Stage 1 PE: Value of state reached - Expectation
        # Note: SARSA-style update often used here, or max Q
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        
        # Update Q-values
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        q_stage1_mf[a1] += learning_rate * delta_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 2: OCI-Driven Stickiness (Perseveration)
This model hypothesizes that high OCI scores manifest as "stickiness" or perseverationâ€”a compulsion to repeat the previous action regardless of the reward outcome. The OCI score scales a `stickiness` parameter added to the Q-values of the previously chosen action. High OCI leads to higher choice repetition.

```python
def cognitive_model2(action_1, state, action_2, reward, oci, model_parameters):
    """
    Model 2: OCI-Driven Stickiness.
    
    This model incorporates a choice autocorrelation (stickiness) parameter that is 
    scaled by the OCI score. High OCI implies a higher tendency to repeat the 
    previous action (compulsive repetition), regardless of reward history.
    
    Parameters:
    learning_rate: [0, 1]
    beta: [0, 10]
    w: [0, 1] - MB/MF mixing weight.
    stickiness_base: [0, 5] - Base tendency to repeat choices.
    
    The effective stickiness added to the Q-value is: stickiness_base * oci_score.
    """
    learning_rate, beta, w, stickiness_base = model_parameters
    n_trials = len(action_1)
    oci_score = oci[0]
    
    effective_stickiness = stickiness_base * oci_score
    
    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    # Track previous choice for stickiness (initialize to -1 for no previous choice)
    prev_action_1 = -1
    
    for trial in range(n_trials):
        # --- Stage 1 Choice ---
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        q_net = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        # Add stickiness bonus if a previous action exists
        q_net_sticky = q_net.copy()
        if prev_action_1 != -1:
            q_net_sticky[prev_action_1] += effective_stickiness
            
        exp_q1 = np.exp(beta * q_net_sticky)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        
        # Update previous action tracker
        prev_action_1 = a1
        
        # --- Stage 2 Choice ---
        s_idx = int(state[trial])
        exp_q2 = np.exp(beta * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Learning ---
        r = reward[trial]
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        q_stage1_mf[a1] += learning_rate * delta_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 3: OCI-Inflated Learning Rate (Hyper-reactivity)
This model suggests that high OCI scores relate to an over-sensitivity to prediction errors, particularly negative ones (fear of failure/imperfection). The learning rate `alpha` is modified by the OCI score. A high OCI score increases the learning rate, making the agent "jumpy" and causing them to change estimates rapidly based on the most recent outcome, rather than integrating over time.

```python
def cognitive_model3(action_1, state, action_2, reward, oci, model_parameters):
    """
    Model 3: OCI-Inflated Learning Rate.
    
    This model proposes that high OCI leads to hyper-reactivity to outcomes. 
    The base learning rate is scaled up by the OCI score. This reflects a cognitive 
    style where the participant weighs recent evidence too heavily (potential 
    anxiety/uncertainty intolerance).
    
    Parameters:
    alpha_base: [0, 1] - Base learning rate.
    beta: [0, 10]
    w: [0, 1] - MB/MF mixing weight.
    lambda_oci: [0, 1] - Sensitivity of learning rate to OCI score.
    
    Effective alpha = alpha_base + (lambda_oci * oci_score).
    (Clipped at 1.0).
    """
    alpha_base, beta, w, lambda_oci = model_parameters
    n_trials = len(action_1)
    oci_score = oci[0]
    
    # Calculate effective learning rate
    # We add a boost based on OCI, capping at 1.0
    learning_rate = min(1.0, alpha_base + (lambda_oci * oci_score))
    
    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    for trial in range(n_trials):
        # --- Stage 1 Choice ---
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        q_net = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        exp_q1 = np.exp(beta * q_net)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        
        # --- Stage 2 Choice ---
        s_idx = int(state[trial])
        exp_q2 = np.exp(beta * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Learning ---
        r = reward[trial]
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        
        # Update using the OCI-modified learning rate
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        q_stage1_mf[a1] += learning_rate * delta_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```
