Here are three cognitive models designed to explain the participant's behavior, incorporating their high OCI-R score into the decision-making mechanics.

### Model 1: Hybrid Learner with OCI-Modulated Model-Based Weight (`cognitive_model1`)

**Concept:** This model posits that individuals with high OCI scores (like this participant) may rely more heavily on habit (Model-Free learning) and less on goal-directed planning (Model-Based learning) due to cognitive rigidity or compulsivity. The `w` parameter, which controls the balance between Model-Based (MB) and Model-Free (MF) systems, is modulated by the OCI score. Specifically, higher OCI reduces the influence of the Model-Based system.

```python
def cognitive_model1(action_1, state, action_2, reward, oci, model_parameters):
    """
    Hybrid RL model where the balance between Model-Based and Model-Free control (w)
    is modulated by the OCI score. High OCI reduces Model-Based influence.
    
    Bounds:
    learning_rate: [0, 1]
    beta: [0, 10]
    w_base: [0, 1] - Baseline mixing weight (0=MF, 1=MB)
    oci_sensitivity: [0, 5] - How strongly OCI reduces MB control
    """
    learning_rate, beta, w_base, oci_sensitivity = model_parameters
    n_trials = len(action_1)
    oci_score = oci[0]
    
    # Calculate effective w: Higher OCI reduces w (pushes towards MF)
    # We clip to ensure it stays between 0 and 1
    w_effective = w_base * (1.0 / (1.0 + oci_sensitivity * oci_score))
    
    # Fixed transition matrix (A->X=0.7, U->Y=0.7)
    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    
    # Q-values
    q_stage1_mf = np.zeros(2)      # Model-free values for stage 1 (Spaceships)
    q_stage2_mf = np.zeros((2, 2)) # Model-free values for stage 2 (Aliens)
    
    for trial in range(n_trials):
        
        # --- Stage 1 Policy ---
        # Model-Based calculation: V_MB(s1) = T * max(Q_stage2)
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        # Hybrid value: Q_net = w * Q_MB + (1-w) * Q_MF
        q_net = w_effective * q_stage1_mb + (1 - w_effective) * q_stage1_mf
        
        # Softmax selection
        exp_q1 = np.exp(beta * q_net)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        # Store probability of observed action
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        
        # --- Stage 2 Policy ---
        s_idx = int(state[trial])
        exp_q2 = np.exp(beta * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Updates ---
        r = reward[trial]
        
        # Stage 2 TD Error (R - Q2)
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        
        # Stage 1 TD Error (Q2 - Q1) - standard SARSA-style update for MF
        # Using the value of the state actually reached
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        q_stage1_mf[a1] += learning_rate * delta_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 2: Perseveration Bias Model (`cognitive_model2`)

**Concept:** Individuals with compulsive tendencies often exhibit "stickiness" or perseverationâ€”repeating the previous choice regardless of reward. This model introduces a perseveration parameter `stickiness` that is scaled by the OCI score. A high OCI score amplifies the tendency to repeat the last action taken at Stage 1, making behavior more rigid and less sensitive to recent outcomes.

```python
def cognitive_model2(action_1, state, action_2, reward, oci, model_parameters):
    """
    Model-Free learner with OCI-modulated perseveration (stickiness).
    High OCI increases the bonus added to the previously chosen option.
    
    Bounds:
    learning_rate: [0, 1]
    beta: [0, 10]
    stickiness_base: [0, 5] - Base tendency to repeat choices
    oci_amp: [0, 5] - How much OCI amplifies the stickiness
    """
    learning_rate, beta, stickiness_base, oci_amp = model_parameters
    n_trials = len(action_1)
    oci_score = oci[0]
    
    # Calculate effective stickiness
    stickiness_effective = stickiness_base + (oci_amp * oci_score)
    
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    last_action_1 = -1 # No previous action for first trial
    
    for trial in range(n_trials):

        # --- Stage 1 Policy ---
        # Add stickiness bonus to Q-values before softmax
        q_stage1_with_stickiness = q_stage1_mf.copy()
        
        if last_action_1 != -1:
            q_stage1_with_stickiness[last_action_1] += stickiness_effective
            
        exp_q1 = np.exp(beta * q_stage1_with_stickiness)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        
        # Update tracker for next trial
        last_action_1 = a1
        
        # --- Stage 2 Policy ---
        s_idx = int(state[trial])
        
        # Standard softmax for stage 2
        exp_q2 = np.exp(beta * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Updates ---
        r = reward[trial]
        
        # Stage 2 Update
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        
        # Stage 1 Update (TD(0))
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        q_stage1_mf[a1] += learning_rate * delta_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 3: Punishment Sensitivity Model (`cognitive_model3`)

**Concept:** Compulsivity is often linked to anxiety and harm avoidance. This model suggests that high OCI individuals learn differently from positive rewards vs. lack of rewards (punishments/omissions). Specifically, this model splits the learning rate into `alpha_pos` and `alpha_neg`. The OCI score shifts the bias such that high OCI individuals have a higher learning rate for negative outcomes (0 coins), reacting more strongly to failure.

```python
def cognitive_model3(action_1, state, action_2, reward, oci, model_parameters):
    """
    Dual learning rate model where OCI increases sensitivity to negative outcomes (0 reward).
    
    Bounds:
    alpha_base: [0, 1] - Base learning rate
    beta: [0, 10]
    neg_bias: [0, 1] - Additional learning rate for negative prediction errors, scaled by OCI
    """
    alpha_base, beta, neg_bias = model_parameters
    n_trials = len(action_1)
    oci_score = oci[0]
    
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    for trial in range(n_trials):
        
        # --- Stage 1 Policy ---
        # Pure Model-Free choice here to isolate learning rate effects
        exp_q1 = np.exp(beta * q_stage1_mf)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        
        # --- Stage 2 Policy ---
        s_idx = int(state[trial])
        exp_q2 = np.exp(beta * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Updates ---
        r = reward[trial]
        
        # Determine effective learning rate based on Prediction Error sign
        # We calculate PE first to know if it's "good" or "bad" news
        pe_stage2 = r - q_stage2_mf[s_idx, a2]
        
        # If PE is negative (outcome worse than expected), boost alpha by OCI * bias
        if pe_stage2 < 0:
            eff_alpha = alpha_base + (neg_bias * oci_score)
        else:
            eff_alpha = alpha_base
            
        # Clip alpha to [0, 1]
        eff_alpha = min(max(eff_alpha, 0.0), 1.0)
        
        # Apply update
        q_stage2_mf[s_idx, a2] += eff_alpha * pe_stage2
        
        # Stage 1 Update
        # Using the same logic: if the Stage 2 value is lower than Stage 1 expectation
        pe_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        
        if pe_stage1 < 0:
            eff_alpha_s1 = alpha_base + (neg_bias * oci_score)
        else:
            eff_alpha_s1 = alpha_base
            
        eff_alpha_s1 = min(max(eff_alpha_s1, 0.0), 1.0)
        
        q_stage1_mf[a1] += eff_alpha_s1 * pe_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```
