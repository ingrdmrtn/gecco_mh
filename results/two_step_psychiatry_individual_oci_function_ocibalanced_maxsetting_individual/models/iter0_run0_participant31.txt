### Model 1: OCI-Modulated Hybrid Control
This model hypothesizes that the balance between Model-Based (goal-directed) and Model-Free (habitual) control is modulated by the participant's obsessive-compulsive symptoms. High OCI scores are often associated with a bias towards habitual control. The parameter `w` (weight for Model-Based values) is a function of the OCI score.

```python
def cognitive_model1(action_1, state, action_2, reward, oci, model_parameters):
    """
    OCI-Modulated Hybrid Model.
    
    The weight (w) determining the balance between Model-Based (MB) and Model-Free (MF) 
    systems is modulated by the OCI score.
    w = w_base + (2 * w_oci_coeff - 1) * oci
    
    Parameters:
    - learning_rate: [0,1] Update rate for MF values.
    - beta: [0,10] Inverse temperature for softmax choice policy.
    - w_base: [0,1] Baseline weight for MB control (intercept).
    - w_oci_coeff: [0,1] Coefficient for OCI modulation on w. 
      Mapped to range [-1, 1] to allow OCI to increase or decrease MB control.
    """
    learning_rate, beta, w_base, w_oci_coeff = model_parameters
    n_trials = len(action_1)
    oci = oci[0]
  
    # Map w_oci_coeff from [0,1] to slope [-1, 1]
    w_slope = 2.0 * w_oci_coeff - 1.0
    w = w_base + w_slope * oci
    w = np.clip(w, 0.0, 1.0)

    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2)) # Dimensions: [State, Action]

    for trial in range(n_trials):
        a1 = int(action_1[trial])
        s_idx = int(state[trial])
        a2 = int(action_2[trial])
        r = reward[trial]

        # --- Policy for the first choice ---
        # Model-Based Value: Expected value over states given transition matrix
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        # Hybrid Value: Weighted mixture of MB and MF
        q_net_1 = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        # Softmax
        exp_q1 = np.exp(beta * (q_net_1 - np.max(q_net_1)))
        probs_1 = exp_q1 / np.sum(exp_q1)
        p_choice_1[trial] = probs_1[a1]

        # --- Policy for the second choice ---
        # Pure Model-Free choice at Stage 2
        q_vals_2 = q_stage2_mf[s_idx]
        exp_q2 = np.exp(beta * (q_vals_2 - np.max(q_vals_2)))
        probs_2 = exp_q2 / np.sum(exp_q2)
        p_choice_2[trial] = probs_2[a2]
  
        # --- Updates ---
        # Update Stage 1 MF value towards Stage 2 value (SARSA-style TD)
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        q_stage1_mf[a1] += learning_rate * delta_stage1
        
        # Update Stage 2 MF value based on reward
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 2: OCI-Modulated Perseveration (Stickiness)
This model hypothesizes that high OCI scores relate to "stickiness" or perseverationâ€”a tendency to repeat the previous choice regardless of the outcome. The model includes a choice autocorrelation parameter (`stickiness`) that is modulated by the OCI score.

```python
def cognitive_model2(action_1, state, action_2, reward, oci, model_parameters):
    """
    OCI-Modulated Stickiness Model.
    
    Adds a 'stickiness' bonus to the previously chosen action in Stage 1.
    The magnitude of this stickiness is modulated by the OCI score.
    
    Parameters:
    - learning_rate: [0,1] Update rate.
    - beta: [0,10] Inverse temperature.
    - w: [0,1] Fixed weight for MB vs MF control.
    - stick_base: [0,1] Baseline stickiness (mapped to [0, 5]).
    - stick_oci: [0,1] Effect of OCI on stickiness (mapped to [-5, 5]).
    """
    learning_rate, beta, w, stick_base, stick_oci = model_parameters
    n_trials = len(action_1)
    oci = oci[0]
    
    # Scale parameters to reasonable logit ranges
    stickiness = (stick_base * 5.0) + ((2.0 * stick_oci - 1.0) * 5.0 * oci)
    
    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    last_a1 = -1 # No previous action for first trial

    for trial in range(n_trials):
        a1 = int(action_1[trial])
        s_idx = int(state[trial])
        a2 = int(action_2[trial])
        r = reward[trial]

        # --- Policy for the first choice ---
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        q_net_1 = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        # Add stickiness to the logits (not the Q-values directly)
        logits = beta * q_net_1
        if last_a1 != -1:
            logits[last_a1] += stickiness
            
        exp_q1 = np.exp(logits - np.max(logits))
        probs_1 = exp_q1 / np.sum(exp_q1)
        p_choice_1[trial] = probs_1[a1]
        
        # Update last action
        last_a1 = a1

        # --- Policy for the second choice ---
        q_vals_2 = q_stage2_mf[s_idx]
        exp_q2 = np.exp(beta * (q_vals_2 - np.max(q_vals_2)))
        probs_2 = exp_q2 / np.sum(exp_q2)
        p_choice_2[trial] = probs_2[a2]
  
        # --- Updates ---
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        q_stage1_mf[a1] += learning_rate * delta_stage1
        
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```
