Here are 3 new cognitive models that incorporate the OCI score to explain the participant's behavior in the two-step task.

### Model 1: OCI-Modulated Learning Rates (Asymmetric Updating)
This model hypothesizes that high OCI scores (associated with compulsivity) might lead to an over-sensitivity to negative outcomes (punishment) or a rigidity in updating values. Here, OCI modulates the learning rate for negative prediction errors (when reward is 0), creating an asymmetry relative to positive updates.

```python
def cognitive_model1(action_1, state, action_2, reward, oci, model_parameters):
    """
    Model 1: OCI-Modulated Asymmetric Learning Rates.
    
    This model posits that OCI affects how strongly participants learn from 
    negative outcomes (zero reward) versus positive outcomes.
    
    The learning rate for negative prediction errors (lr_neg) is modulated by OCI:
    lr_neg = lr_base * (1 + oci_sensitivity * oci)
    The learning rate for positive prediction errors is just lr_base.
    
    Parameters:
    lr_base: [0, 1] - Base learning rate for positive outcomes.
    beta: [0, 10] - Inverse temperature for softmax choice.
    oci_sensitivity: [0, 1] - Scaling factor for how much OCI boosts negative learning.
    w: [0, 1] - Mixing weight for Model-Based (1) vs Model-Free (0) control.
    """
    lr_base, beta, oci_sensitivity, w = model_parameters
    n_trials = len(action_1)
    current_oci = oci[0]

    # Calculate asymmetric learning rates
    # If OCI is high, learning from failure might be amplified (fear of failure/checking)
    lr_pos = lr_base
    lr_neg = np.clip(lr_base * (1.0 + oci_sensitivity * current_oci), 0.0, 1.0)

    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)

    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))

    for trial in range(n_trials):
        # --- Stage 1 Choice ---
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        # Integrated Q-value
        q_net = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        exp_q1 = np.exp(beta * q_net)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        
        # --- Stage 2 Choice ---
        s_idx = int(state[trial])
        
        exp_q2 = np.exp(beta * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Learning ---
        r = reward[trial]
        
        # Calculate Prediction Errors
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        
        # Apply asymmetric learning rates based on the sign of the PE
        # Note: In this simple task, reward is 0 or 1.
        # If r=1, PE is usually positive. If r=0, PE is negative.
        # We use the outcome itself to decide the rate for simplicity in this binary context.
        
        current_lr = lr_pos if r > 0.5 else lr_neg
        
        q_stage2_mf[s_idx, a2] += current_lr * delta_stage2
        q_stage1_mf[a1] += current_lr * delta_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 2: OCI-Driven Choice Stickiness (Perseveration)
This model assumes that compulsivity (OCI) manifests as "stickiness" or perseverationâ€”a tendency to repeat the previous action regardless of the reward outcome.

```python
def cognitive_model2(action_1, state, action_2, reward, oci, model_parameters):
    """
    Model 2: OCI-Driven Choice Stickiness.
    
    This model introduces a choice stickiness parameter that is directly 
    scaled by the OCI score. High OCI leads to higher perseveration 
    (repeating the previous Stage 1 choice).
    
    Stickiness bonus = stick_base + (stick_oci_scale * oci)
    
    Parameters:
    learning_rate: [0, 1] - Learning rate for Q-values.
    beta: [0, 10] - Inverse temperature.
    w: [0, 1] - MB/MF mixing weight.
    stick_oci_scale: [0, 5] - How much OCI increases stickiness.
    """
    learning_rate, beta, w, stick_oci_scale = model_parameters
    n_trials = len(action_1)
    current_oci = oci[0]
    
    # Calculate stickiness based on OCI
    # We assume a base stickiness of 0 for simplicity, driven purely by OCI here
    # or one could add a base param. Here we focus on the OCI contribution.
    stickiness = stick_oci_scale * current_oci

    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)

    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    # Track previous choice for stickiness
    last_action_1 = -1

    for trial in range(n_trials):
        # --- Stage 1 Choice ---
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        q_net = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        # Add stickiness bonus to the Q-values before softmax
        q_net_stick = q_net.copy()
        if last_action_1 != -1:
            q_net_stick[last_action_1] += stickiness
            
        exp_q1 = np.exp(beta * q_net_stick)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        last_action_1 = a1 # Update for next trial
        
        # --- Stage 2 Choice ---
        s_idx = int(state[trial])
        
        exp_q2 = np.exp(beta * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Learning ---
        r = reward[trial]
        
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        q_stage1_mf[a1] += learning_rate * delta_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 3: OCI-Modulated Temperature (Decision Noise)
This model proposes that high OCI is associated with reduced decision noise (higher beta), reflecting a more rigid or deterministic application of the learned values (whether correct or habitual). Alternatively, it could capture anxiety-driven randomness. Here we parameterize it so `beta` scales with OCI.

```python
def cognitive_model3(action_1, state, action_2, reward, oci, model_parameters):
    """
    Model 3: OCI-Modulated Inverse Temperature.
    
    This model assumes that the exploration/exploitation trade-off (beta)
    is influenced by OCI. 
    
    effective_beta = beta_base * (1 + beta_oci_scale * oci)
    
    If beta_oci_scale is positive, high OCI makes behavior more deterministic (rigid).
    
    Parameters:
    learning_rate: [0, 1] - Learning rate.
    beta_base: [0, 10] - Baseline inverse temperature.
    beta_oci_scale: [0, 5] - Scaling factor for OCI's effect on beta.
    w: [0, 1] - MB/MF mixing weight.
    """
    learning_rate, beta_base, beta_oci_scale, w = model_parameters
    n_trials = len(action_1)
    current_oci = oci[0]
    
    # Modulate beta
    beta_eff = beta_base * (1.0 + beta_oci_scale * current_oci)

    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)

    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))

    for trial in range(n_trials):
        # --- Stage 1 Choice ---
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        q_net = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        exp_q1 = np.exp(beta_eff * q_net)
        probs_1 = exp_q1 / np.sum(exp_q1)
        
        a1 = int(action_1[trial])
        p_choice_1[trial] = probs_1[a1]
        
        # --- Stage 2 Choice ---
        s_idx = int(state[trial])
        
        exp_q2 = np.exp(beta_eff * q_stage2_mf[s_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        
        a2 = int(action_2[trial])
        p_choice_2[trial] = probs_2[a2]
        
        # --- Learning ---
        r = reward[trial]
        
        delta_stage2 = r - q_stage2_mf[s_idx, a2]
        delta_stage1 = q_stage2_mf[s_idx, a2] - q_stage1_mf[a1]
        
        q_stage2_mf[s_idx, a2] += learning_rate * delta_stage2
        q_stage1_mf[a1] += learning_rate * delta_stage1

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```
