### Model 1: Reactive Model-Free Retreat
This model hypothesizes that anxiety causes a dynamic shift in control strategy. Specifically, when an anxious participant experiences a "threat" (a loss/0 coins), they retreat from computationally expensive Model-Based planning to habitual Model-Free control on the subsequent trial.

```python
def cognitive_model1(action_1, state, action_2, reward, stai, model_parameters):
    """
    Reactive Model-Free Retreat Model.
    
    Hypothesis: Anxiety modulates the arbitration between Model-Based (MB) and 
    Model-Free (MF) control dynamically. Following a loss (0 reward), anxious 
    individuals 'retreat' to MF control (habit) to save cognitive resources, 
    reducing the weight (w) of the MB system for the next trial.
    
    Parameters:
    - learning_rate: [0, 1] Standard Q-learning rate.
    - beta: [0, 10] Inverse temperature.
    - w_base: [0, 1] Baseline weight for model-based control.
    - retreat_factor: [0, 1] How much 'w' decreases after a loss, scaled by STAI.
    """
    learning_rate, beta, w_base, retreat_factor = model_parameters
    n_trials = len(action_1)
    stai_score = stai[0]

    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    last_reward = 1 # Initialize as a 'win' so no retreat on first trial

    for trial in range(n_trials):

        # policy for the first choice
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        # Dynamic w calculation
        current_w = w_base
        if last_reward == 0:
            # If previous trial was a loss, reduce MB weight based on anxiety
            current_w = w_base - (retreat_factor * stai_score)
            # Ensure w stays within bounds [0, 1]
            if current_w < 0: current_w = 0
            if current_w > 1: current_w = 1

        q_net = current_w * q_stage1_mb + (1 - current_w) * q_stage1_mf
        
        exp_q1 = np.exp(beta * q_net)
        probs_1 = exp_q1 / np.sum(exp_q1)
        p_choice_1[trial] = probs_1[action_1[trial]]
        state_idx = state[trial]

        # policy for the second choice
        exp_q2 = np.exp(beta * q_stage2_mf[state_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        p_choice_2[trial] = probs_2[action_2[trial]]
  
        delta_stage1 = q_stage2_mf[state_idx, action_2[trial]] - q_stage1_mf[action_1[trial]]
        q_stage1_mf[action_1[trial]] += learning_rate * delta_stage1
        
        delta_stage2 = reward[trial] - q_stage2_mf[state_idx, action_2[trial]]
        q_stage2_mf[state_idx, action_2[trial]] += learning_rate * delta_stage2
        
        last_reward = reward[trial]

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 2: Anxiety-Driven Win-Stay Bias
This model refines the concept of perseveration. Instead of blindly repeating actions, anxious participants are hypothesized to cling specifically to "safe" actionsâ€”those that were previously rewarded.

```python
def cognitive_model2(action_1, state, action_2, reward, stai, model_parameters):
    """
    Anxiety-Driven Win-Stay Bias Model.
    
    Hypothesis: Anxiety increases the tendency to repeat a Stage 1 choice ONLY if 
    it resulted in a reward on the previous trial. This represents a 'safety seeking' 
    heuristic where anxious agents cling to proven safe options.
    
    Parameters:
    - learning_rate: [0, 1] Standard Q-learning rate.
    - beta: [0, 10] Inverse temperature.
    - w: [0, 1] Weight for model-based control.
    - win_stickiness: [0, 5] Additional value added to the previously chosen action 
      if it was rewarded, scaled by STAI.
    """
    learning_rate, beta, w, win_stickiness = model_parameters
    n_trials = len(action_1)
    stai_score = stai[0]

    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))
    
    last_action_1 = -1
    last_reward = 0

    for trial in range(n_trials):

        # policy for the first choice
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        
        q_net = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        # Apply stickiness bonus only if the last trial was a Win
        if last_action_1 != -1 and last_reward == 1:
            q_net[last_action_1] += (win_stickiness * stai_score)

        exp_q1 = np.exp(beta * q_net)
        probs_1 = exp_q1 / np.sum(exp_q1)
        p_choice_1[trial] = probs_1[action_1[trial]]
        
        last_action_1 = action_1[trial]
        state_idx = state[trial]

        # policy for the second choice
        exp_q2 = np.exp(beta * q_stage2_mf[state_idx])
        probs_2 = exp_q2 / np.sum(exp_q2)
        p_choice_2[trial] = probs_2[action_2[trial]]
  
        delta_stage1 = q_stage2_mf[state_idx, action_2[trial]] - q_stage1_mf[action_1[trial]]
        q_stage1_mf[action_1[trial]] += learning_rate * delta_stage1
        
        delta_stage2 = reward[trial] - q_stage2_mf[state_idx, action_2[trial]]
        q_stage2_mf[state_idx, action_2[trial]] += learning_rate * delta_stage2
        
        last_reward = reward[trial]

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```

### Model 3: Anxiety-Induced Decision Noise (Lapse)
This model posits that anxiety acts as a distractor, introducing random "noise" or "lapses" into the decision process. High anxiety increases the probability of making a random choice, regardless of the calculated value.

```python
def cognitive_model3(action_1, state, action_2, reward, stai, model_parameters):
    """
    Anxiety-Induced Decision Noise (Lapse) Model.
    
    Hypothesis: Anxiety acts as a distractor that decouples choice from value. 
    The agent follows the softmax policy with probability (1 - epsilon) and 
    chooses randomly with probability (epsilon). Epsilon is driven by STAI.
    
    Parameters:
    - learning_rate: [0, 1] Standard Q-learning rate.
    - beta: [0, 10] Inverse temperature for the value-based portion of choice.
    - w: [0, 1] Weight for model-based control.
    - noise_factor: [0, 1] Scaling factor for STAI to determine the lapse rate (epsilon).
    """
    learning_rate, beta, w, noise_factor = model_parameters
    n_trials = len(action_1)
    stai_score = stai[0]
    
    # Calculate epsilon (lapse rate) based on anxiety
    epsilon = noise_factor * stai_score
    # Ensure epsilon is within [0, 1]
    if epsilon > 1.0: epsilon = 1.0

    transition_matrix = np.array([[0.7, 0.3], [0.3, 0.7]])
    p_choice_1 = np.zeros(n_trials)
    p_choice_2 = np.zeros(n_trials)
    q_stage1_mf = np.zeros(2)
    q_stage2_mf = np.zeros((2, 2))

    for trial in range(n_trials):

        # policy for the first choice
        max_q_stage2 = np.max(q_stage2_mf, axis=1)
        q_stage1_mb = transition_matrix @ max_q_stage2
        q_net = w * q_stage1_mb + (1 - w) * q_stage1_mf
        
        exp_q1 = np.exp(beta * q_net)
        probs_1_softmax = exp_q1 / np.sum(exp_q1)
        
        # Mixture of Softmax and Random Uniform (0.5 for 2 choices)
        probs_1 = (1 - epsilon) * probs_1_softmax + epsilon * 0.5
        p_choice_1[trial] = probs_1[action_1[trial]]
        
        state_idx = state[trial]

        # policy for the second choice
        exp_q2 = np.exp(beta * q_stage2_mf[state_idx])
        probs_2_softmax = exp_q2 / np.sum(exp_q2)
        
        # Apply noise to second stage as well
        probs_2 = (1 - epsilon) * probs_2_softmax + epsilon * 0.5
        p_choice_2[trial] = probs_2[action_2[trial]]
  
        delta_stage1 = q_stage2_mf[state_idx, action_2[trial]] - q_stage1_mf[action_1[trial]]
        q_stage1_mf[action_1[trial]] += learning_rate * delta_stage1
        
        delta_stage2 = reward[trial] - q_stage2_mf[state_idx, action_2[trial]]
        q_stage2_mf[state_idx, action_2[trial]] += learning_rate * delta_stage2
        

    eps = 1e-10
    log_loss = -(np.sum(np.log(p_choice_1 + eps)) + np.sum(np.log(p_choice_2 + eps)))
    return log_loss
```
